# IRMCTracker

A discord bot  for tracking Iranian Minecraft servers and showing the statistics of them

The old Python source code is now replaced with this new TypeScript project.


## Installation

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```
